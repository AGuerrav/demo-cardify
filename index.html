Cardify

Una libreria que te permite sobreponer texto en la imagen, al pasar el mouse sobre ella

Tambien funciona con archivos de extension .gif

Modo de uso

Download

github

-Descarga el ropositorio github 

            -Enlaza jQuery en tu proyecto y el archivo main.js en la carpeta lib 

            -Usa la clase 'imgHover' en el contenerdor de tu imagen

            <script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.4/jquery.min.js"></script>
<script type="text/javascript" src="./lib/main.js"></script>
<div class='imgHover'><img>src="ruta-imagen" alt="frase que se muestra al pasar el mouse"</img></div>
