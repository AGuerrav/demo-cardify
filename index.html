Cardify

Una librería que te permite sobreponer texto en la imagen, al pasar el mouse sobre ella

También funciona con archivos de extensión .gif

Modo de uso

Download

github

-Descarga el repositorio github 

            -Enlaza en tu proyecto el archivo main.js en la carpeta lib 

            -Usa la clase 'imgHover' en el contenedor de tu imagen
            <script type="text/javascript" src="./lib/main.js"></script>
<div class='imgHover'><img>src="ruta-imagen" alt="frase que se muestra al pasar el mouse"</img></div>
